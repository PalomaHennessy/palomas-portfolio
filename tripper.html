Tripper

Working as a member of a development team is never easy. This project was built by myself and three other students. We attempted to create a way for the user to decide where to stop from start to stop, for food. Although this isn't my favorite design, it was a great learning experience. I learned more about give and take between team members, what battles to fight and how to really work together to finish a project in a given amount of time.

Technologies Used

  • Ruby on Rails
  • OmniAuth
  • OAuth
  • Tubular
  • HTML/CSS/Javascript
  • Materialize
  • Google Maps API
  • Google Places API
