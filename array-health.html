Designed with: AdobeXD
Project Details

  • Client: Array Health (Get Insured)
  • Date: July, 2016

At Array Health (now Get Insured) I worked to improve the way people shop for and enroll in health insurance. As a visual designer I found it particularly challenging and interesting to try and design around insurance companies requests while working to drive the platform forward.

One of my main projects was re-designing the enrollment process. My main goal when working on this project, was to simplify the experience. I did this by breaking the process into more distict steps, showing the user where they were in the enrollment process, and using color to both direct the user and display information in a more understandable way.
